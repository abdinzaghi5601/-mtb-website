# Product Images Directory

## Adding Product Images

Place your product images in this directory with the following naming convention:

### Disc Cutters
- `disc-cutter-17.jpg` - Main image for 17-inch disc cutter
- `disc-cutter-17-side.jpg` - Side view
- `disc-cutter-17-detail.jpg` - Detail/close-up view
- `disc-cutter-19.jpg` - Main image for 19-inch disc cutter
- `disc-cutter-19-side.jpg` - Side view

### Bearings
- `main-bearing.jpg` - Main bearing image
- `main-bearing-assembly.jpg` - Assembly view
- `thrust-bearing.jpg` - Thrust bearing image

### Spare Parts
- `cutter-ring.jpg` - Cutter ring
- `cutter-ring-installed.jpg` - Installed view
- `seal-kit.jpg` - Seal kit
- `cutter-block.jpg` - Cutter block assembly
- `gripper-pad.jpg` - Gripper shoe pad
- `conveyor-blade.jpg` - Screw conveyor blade

## Image Requirements

- **Format**: JPG or PNG
- **Recommended Size**: 1200x800 pixels (or similar aspect ratio)
- **File Size**: Optimize images to under 500KB for fast loading
- **Quality**: High resolution for zoom functionality

## Tips

1. Use consistent lighting and background for professional appearance
2. Include multiple angles (front, side, detail) for better product showcase
3. Ensure images are well-lit and in focus
4. Consider using a white or neutral background

## Note

If you don't have images yet, the website will show placeholder images. You can add your actual product photos later by replacing the placeholder paths in the product data files.
